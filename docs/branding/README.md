These are *data*, not code — they belong in configuration, not in components.

- **Location:** Adair Village, Oregon (Pacific Northwest)
- **Service area:** Benton, Linn, Marion, Polk counties

## Visual direction (for the redesign)
- Premium craftsmanship + warmth + trust + attention to detail
- Large photography, natural spacing, excellent typography
- Palette: warm whites, wood tones, forest greens, muted blues, soft shadows
- Minimal/subtle motion only

## Assets
- Client-owned project photos: download into `public/images/originals/` then generate
  WebP/AVIF/responsive + blur placeholders. Never hot-link Google Site URLs.
- Logo / mark: store in `public/images/logos/`.

## Reuse rule
Brand colors, copy, and images are referenced via `config/` and image metadata.
Swapping to contractor #2 means changing `config/` + `public/images/`, not code.
