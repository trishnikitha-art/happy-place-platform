# Oregon Regional Authority Structure

## Objective

Create regional profiles for Oregon service areas that account for travel, permit tendencies, inspection timelines, demand, weather, fuel costs, and supplier distances. These factors don't change prices directly—they affect planning confidence and operational considerations.

## Philosophy

Regional factors are multipliers for operational complexity, not price multipliers. They affect:
- Planning confidence (wider ranges for complex regions)
- Scheduling considerations
- Material availability
- Permit timelines

## JSON Structure

```typescript
interface RegionalProfile {
  region: string;
  county: string;
  
  travel: {
    multiplier: number;
    typicalTime: string;
    considerations: string[];
  };
  
  permits: {
    strictness: "relaxed" | "standard" | "stricter";
    processingTime: string;
    inspectionScheduling: string;
    feeLevel: "lower" | "moderate" | "higher";
    requirements: string;
  };
  
  inspections: {
    foundation: string;
    framing: string;
    final: string;
    totalCycle: string;
  };
  
  demand: {
    level: "low" | "medium" | "high";
    seasonality: string;
    competition: string;
    projectTypes: string;
  };
  
  weather: {
    rainfall: string;
    winter: string;
    summer: string;
    constructionSeason: string;
  };
  
  fuel: {
    cost: string;
    impact: number;
  };
  
  suppliers: {
    lumber: string;
    hardware: string;
    specialty: string;
    concrete: string;
  };
  
  confidenceImpact: number;
}
```

## Usage in Planning Engine

Regional profiles are used to:
1. Adjust planning confidence based on regional complexity
2. Add operational considerations to estimate output
3. Provide context for scheduling timelines
4. Account for travel and logistics in planning ranges

Regional factors do not directly multiply prices—they affect the confidence score, which widens or narrows the planning range.

## Monitoring

- **Accuracy**: Track planning vs. actual by region
- **Confidence Calibration**: Adjust confidence impact based on accuracy
- **Permit Changes**: Monitor permit process changes by region
- **Supplier Changes**: Track supplier distance changes
- **Seasonal Variations**: Monitor seasonal factor accuracy

## Future Enhancements

- **Real-Time Traffic**: Integrate with traffic APIs for dynamic travel estimates
- **Weather Integration**: Real-time weather for seasonal adjustments
- **Permit API Integration**: Direct integration with permit systems for timelines
- **Supplier API Integration**: Real-time supplier availability and pricing
- **Sub-Regional Profiles**: More granular profiles within larger regions
